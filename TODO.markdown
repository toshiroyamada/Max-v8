### Arguments

Type: Array  
Name: jsarguments

    jsarguments[0]      // returns filename
    jsarguments[1]      // first argument
    jsarguments.length  // returns size of the array
                        // (always 1 or more)

Arguments are passed in through the external, after specifying the filename.
